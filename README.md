# 🏗️ Autonomous Infrastructure Inspection

This project uses **YOLO11-Segmentation** to detect cracks, potholes, and corrosion 
from drone imagery. It is part of my final project for **ITSOLERA**.

### 🚀 Features
- **Multi-class Segmentation:** Detects 3 types of defects in one pass.
- **Cloud Inference:** Powered by Roboflow API.
- **Maintenance Reporting:** Automatically calculates defect priority.

### 📊 Performance
- **Corrosion Accuracy:** 99.0% mAP
- **Overall mAP:** 53.1%
